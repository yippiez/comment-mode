# comment-mode

A review-based workflow to work with coding agents.

It provides:
- keyboard-first file/code navigation
- inline agent prompt rows
- in-app search modal with previews
- multiple visual themes

## Run

```bash
bun install
bun dev
```

## Themes

- Vague
- OpenCode
- Tokyo Night
- Soda
